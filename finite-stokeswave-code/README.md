Analytic continuation of Stokes wave
====================================

Contributors
-------------
 * Crew, S. C. (Summer 2015)
 * Johnson-Llambias, Y. 
 * Trinh, P. H.

Code to solve for finite-depth Stokes waves (series approximation and boundary integral) and its analytic continuation. This will re-use many of the old codes dedicated to the case of infinite depth.

1. To push to the bitbucket repository, use

    hg commit -A -m 'New log message'

    hg push https://[user]@bitbucket.org/trinh/finite-stokeswave-code
    
    or 
    
    hg push ssh://hg@bitbucket.org/trinh/finite-stokeswave-code
